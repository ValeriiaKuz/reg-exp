Regular expressions

Regular expressions are patterns used to match character combinations in strings.
In JS, regular expressions are objects

RegExp

How to create?

/.../ notation

Benefits

  • Short syntax

Cons

  • Do not allow any variable insertions

new RegExp(...) notation

Benefits

  • Allow variable insertions

Cons

  • Long syntax
How does it look like?
